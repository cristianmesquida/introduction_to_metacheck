## Introduction to Metacheck

This repository contains the materials used for the workshop "Introduction to Metacheck"

## How to use the repository

The file script.Rmd contains the instructions to use Metacheck and their corresponding modules.

## Project structure
```rmarkdown
┌─ script.Rmd: Contains instructions for using Metacheck and demosntration of its modules.
├─ pdf_folder: Manually downloaded PDF articles used as examples for batch processing.
├─ pdf_folder_2: Automatically donwloaded PDF articles used as examples for batch processing.
├─ xml_folder: XML articles converted from pdf_folder, used as examples for batch processing.
├─ collabra.Rds: contains XML articles from xml_folder.
├─ manuscript.pdf: PDF example article.
├─ manuscript.xml: XML example article converted from manuscript.pdf.
├─ bv9ds/: folder containing example files downloaded from OSF repository using Metacheck
└─ ek2gp/: folder containing example files downloaded from OSF repository using Metacheck
```
